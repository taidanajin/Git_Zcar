# 🤖 Инструкция по настройке Telegram Monitor Bot

Этот бот будет отслеживать все новые сообщения в вашем Telegram канале и **автоматически добавлять их в раздел "Объявления" на сайте**.

---

## 📋 Что нужно для работы

1. **Токен бота** - получите у @BotFather
2. **ID канала** - уже настроен (-3261646428)
3. **База данных** - создается автоматически при первом запуске main.py

---

## 🔧 Шаг 1: Создание бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Придумайте имя для бота (например: "Царь Monitor Bot")
4. Придумайте username для бота (должен заканчиваться на "bot", например: "tsar_monitor_bot")
5. **Скопируйте токен**, который выдаст BotFather (выглядит примерно так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

---

## ⚙️ Шаг 2: Настройка бота

### Вариант А: Через переменные окружения (рекомендуется)

Откройте командную строку и выполните:

```cmd
set TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
```

**Важно:** Эта переменная действует только в текущей сессии. Для постоянной установки:

1. Нажмите Win+R
2. Введите `sysdm.cpl` и нажмите Enter
3. Перейдите на вкладку "Дополнительно"
4. Нажмите "Переменные среды"
5. Создайте новую переменную:
   - `TELEGRAM_BOT_TOKEN` = ваш токен

### Вариант Б: Редактирование файла

Откройте файл `telegram_monitor_bot.py` и найдите строку:

```python
BOT_TOKEN = os.environ.get('TELEGRAM_BOT_TOKEN', 'YOUR_BOT_TOKEN_HERE')
```

Замените на:

```python
BOT_TOKEN = os.environ.get('TELEGRAM_BOT_TOKEN', '1234567890:ABCdefGHIjklMNOpqrsTUVwxyz')
```

---

## 🔐 Шаг 3: Добавление бота в канал

1. Откройте ваш Telegram канал
2. Нажмите на название канала → "Управление каналом"
3. Выберите "Администраторы" → "Добавить администратора"
4. Найдите вашего бота по username
5. Дайте боту права:
   - ✅ Просмотр сообщений (обязательно!)
   - Остальные права можно отключить

---

## 🚀 Шаг 4: Запуск бота

### Способ 1: Через bat-файл (проще)

Просто запустите файл `start_monitor_bot.bat` двойным кликом

### Способ 2: Через командную строку

```cmd
python telegram_monitor_bot.py
```

---

## ✅ Проверка работы

После запуска бот должен:

1. Показать сообщение "🤖 Telegram Monitor Bot запущен"
2. Начать отслеживать новые сообщения в канале
3. Автоматически добавлять каждое новое сообщение в раздел "Объявления" на сайте

---

## 📝 Тестирование

1. Запустите бота
2. Опубликуйте тестовое сообщение в канале
3. Через несколько секунд проверьте раздел "Объявления" на сайте (http://127.0.0.1:5000/obj)
4. Новое объявление должно появиться автоматически!

---

## 🛑 Остановка бота

Нажмите `Ctrl+C` в окне командной строки, где запущен бот

---

## 🔧 Настройки

В файле `telegram_monitor_bot.py` можно изменить:

- `CHANNEL_ID` - ID канала для мониторинга (по умолчанию: -3261646428)
- `DATABASE_PATH` - путь к базе данных (по умолчанию: instance/main.db)
- `LAST_MESSAGE_FILE` - файл для хранения последнего ID сообщения

---

## ❓ Частые проблемы

### Бот не получает сообщения из канала

**Решение:**
- Убедитесь, что бот добавлен в канал как администратор
- Проверьте, что у бота есть право "Просмотр сообщений"
- Проверьте правильность ID канала

### Объявления не добавляются на сайт

**Решение:**
- Убедитесь, что база данных существует (запустите main.py)
- Проверьте путь к базе данных в настройках
- Посмотрите сообщения об ошибках в консоли

### Ошибка "Unauthorized"

**Решение:**
- Проверьте правильность токена бота
- Убедитесь, что токен скопирован полностью

### Ошибка "База данных не найдена"

**Решение:**
- Сначала запустите main.py хотя бы один раз
- Убедитесь, что создана папка instance/ с файлом main.db

---

## 📊 Дополнительные возможности

Бот автоматически:
- ✅ Извлекает заголовок из первой строки сообщения
- ✅ Сохраняет полный текст сообщения
- ✅ Запоминает дату публикации
- ✅ Сохраняет ID сообщения из Telegram
- ✅ Не дублирует уже добавленные сообщения
- ✅ Пропускает слишком короткие сообщения (менее 5 символов)
