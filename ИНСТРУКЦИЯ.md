# Быстрая настройка интеграции с Telegram

## 📋 Что нужно сделать

### 1. Создать Telegram бота

1. Откройте Telegram
2. Найдите **@BotFather**
3. Отправьте команду `/newbot`
4. Придумайте имя и username для бота
5. **Скопируйте токен** (выглядит как `123456789:ABCdefGHI...`)

### 2. Добавить бота в канал

1. Откройте ваш канал: https://web.telegram.org/k/#-3261646428
2. Настройки канала → Администраторы → Добавить администратора
3. Найдите вашего бота и добавьте его
4. Дайте права на **чтение сообщений**

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить токен бота

**Windows (PowerShell):**
```powershell
$env:TELEGRAM_BOT_TOKEN="ваш_токен_от_BotFather"
```

**Windows (CMD):**
```cmd
set TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
```

**Linux/Mac:**
```bash
export TELEGRAM_BOT_TOKEN="ваш_токен_от_BotFather"
```

### 5. Запустить настройку

```bash
python setup_telegram.py
```

Этот скрипт:
- Создаст таблицу для объявлений в БД
- Добавит примеры объявлений
- Проверит подключение к Telegram

### 6. Синхронизировать сообщения

```bash
python sync_telegram.py
```

Этот скрипт загрузит все сообщения из вашего Telegram канала в базу данных.

### 7. Запустить приложение

```bash
python main.py
```

Откройте в браузере: http://127.0.0.1:5000/obj

## 🔄 Автоматическая синхронизация

Чтобы объявления обновлялись автоматически, запускайте `sync_telegram.py` периодически:

**Windows (Task Scheduler):**
1. Откройте "Планировщик заданий"
2. Создайте новое задание
3. Триггер: каждые 5-10 минут
4. Действие: запуск `python sync_telegram.py`

**Linux (cron):**
```bash
# Редактировать crontab
crontab -e

# Добавить строку (каждые 5 минут)
*/5 * * * * cd /путь/к/проекту && python sync_telegram.py
```

## ✅ Проверка работы

1. Отправьте сообщение в Telegram канал
2. Запустите `python sync_telegram.py`
3. Откройте http://127.0.0.1:5000/obj
4. Ваше сообщение должно появиться в объявлениях!

## 📝 Примечания

- Первая строка сообщения становится заголовком
- Сообщения короче 10 символов игнорируются
- Дубликаты не добавляются повторно
- ID канала уже настроен: `-3261646428`

## ❓ Проблемы?

**Объявления не появляются:**
- Проверьте, что бот добавлен в канал как администратор
- Проверьте токен бота: `echo $env:TELEGRAM_BOT_TOKEN`
- Запустите `python sync_telegram.py` и посмотрите на ошибки

**Ошибка "Bot token not configured":**
- Установите переменную окружения TELEGRAM_BOT_TOKEN
- Перезапустите терминал после установки

**Нет новых сообщений:**
- Telegram API возвращает только последние ~100 обновлений
- Для старых сообщений используйте другие методы API
