# 🎭 Клуб "Царь" - Сайт мафия-клуба

Веб-приложение для мафия-клуба с интеграцией Telegram для автоматической публикации объявлений.

## ✨ Возможности

- 🏠 Главная страница с информацией о клубе
- 📢 Автоматическая синхронизация объявлений из Telegram канала
- 📜 Подробные правила игры
- 🗺️ Интерактивная карта с местоположением клуба
- 👥 Система авторизации пользователей
- 🌓 Переключение светлой/тёмной темы
- 🎨 Стильный дизайн в карточной тематике

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка Telegram бота

Следуйте инструкциям в файле `БЫСТРЫЙ_СТАРТ.txt` или `ИНСТРУКЦИЯ.md`

Кратко:
1. Создайте бота через @BotFather
2. Добавьте бота в канал как администратора
3. Установите токен: `set TELEGRAM_BOT_TOKEN=ваш_токен`

### 3. Инициализация базы данных

```bash
python setup_telegram.py
```

### 4. Синхронизация объявлений

```bash
python sync_telegram.py
```

### 5. Запуск приложения

```bash
python main.py
```

Откройте в браузере: http://127.0.0.1:5000

## 🗄️ База данных

### Таблица User
- `id` - ID пользователя
- `username` - Логин
- `password` - Хеш пароля
- `admin` - Флаг администратора

### Таблица Announcement
- `id` - ID объявления
- `title` - Заголовок
- `content` - Текст объявления
- `date` - Дата публикации
- `telegram_message_id` - ID сообщения из Telegram

## 🔄 Автоматическая синхронизация

Для автоматического обновления объявлений настройте периодический запуск `sync_telegram.py`:

**Windows (Task Scheduler):**
- Создайте задание с запуском каждые 5-10 минут

**Linux (cron):**
```bash
*/5 * * * * cd /путь/к/проекту && python sync_telegram.py
```
